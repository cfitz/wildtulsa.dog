CENTER-TBI Neurobotmore_vert

Project Home

Traumatic Brain Injury, is a leading cause of death and disability, with an expected 50-60 million new cases annually. CENTER-TBI aims to better characterize TBI and identify the most effective treatments and improve outcomes via comparative-effectiveness studies. Neurbot is a data collection tool that allows researchers to normalize and export data collected by the study. It is written in ReactJS using Next.js framework.
